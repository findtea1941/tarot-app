# 塔罗模块 — 用户流程 V1

## 目标
用户可以：新建案例 → 录入基础信息 → 录入牌阵（支持逆位）→ 补充信息（ACE/宫廷牌行星 + 指示牌）→ 自动生成统筹分析 → 填写解读 → 保存 → 在案例库复盘/编辑。

---

## Tab / 页面
- Tab：塔罗、案例库（雷诺曼先占位）
- 页面：
  - 塔罗首页（入口）
  - 新建案例：基础信息页
  - 牌阵录入页（支持返回改基础信息）
  - 补充信息弹窗（可跳过）
  - 分析页（统筹表格 + 解读 + 保存）
  - 案例库列表页
  - 案例详情页（复盘编辑）

---

## 主流程

### Step 1：塔罗首页
- 按钮：新建案例 → 进入基础信息页
- 按钮：进入案例库 → 进入案例库列表

### Step 2：基础信息页
字段：
- question（必填）
- background（可选）
- category（必选：情感/事业/学业/其他）
- drawTime（必选，精确到分钟）
- spreadType（必选：六芒星/四元素/二择一/身心灵/圣三角/时间流/无牌阵）
按钮：
- 下一步：进入牌阵
系统：
- 校验必填项，成功后创建 case 草稿并进入牌阵录入页；失败提示错误。

### Step 3：牌阵录入页
内容：
- 顶部只读展示：基础信息
- 按 spreadType 渲染 N 个位置（positionIndex + positionName + 输入框）
输入规则：
- “牌名”=正位
- “牌名-”=逆位（末尾 -）
按钮：
- 返回修改案例信息（回到基础信息页，数据保留）
- 确定
系统（点确定）：
- 校验缺牌、牌名有效性、格式
- 成功：弹出补充信息弹窗
- 失败：停留本页并定位提示

### Step 4：补充信息弹窗（可跳过）
内容：
- 若出现 ACE/宫廷牌：显示待补充列表，每张提供行星下拉
- 指示牌输入框：多张用 “;” 分隔，支持逆位“-”
按钮：
- 确定（保存补充信息，进入分析页）
- 跳过（不填也可，进入分析页）
系统：
- 如补充输入有误，在弹窗内提示并阻止关闭

### Step 5：分析页
内容：
- 左侧：基础信息 + 牌阵布局展示（逆位带“-”）
- 右侧：统筹分析表格（自动生成） + 用户解读输入框
逆位规则（必须）：
- 展示：逆位牌名显示为 “牌名-”
- 计算：若牌数字为 n，则逆位计入值为 -n
按钮：
- 保存（保存后停留）
- 保存并返回案例库（保存后跳转案例库）
标题规则：
- title = YYYY-MM-DD | category | question

### Step 6：案例库列表
- 列表展示、排序（默认最新）、分类筛选、关键词搜索
- 点击进入案例详情

### Step 7：案例详情（复盘）
- 展示：基础信息、牌阵、统筹表格、指示牌、用户解读
- 可编辑：用户解读、指示牌标题
- 保存修改
